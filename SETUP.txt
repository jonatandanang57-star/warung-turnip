## 🚀 Langkah Setup Cepat:

1. **Copy semua file ke:** `C:\xampp\htdocs\PROJECT SQL\`

2. **Buat database MySQL:**
   - Buka phpMyAdmin: http://localhost/phpmyadmin
   - Buat database baru: `toko_management`
   - Import file: `database.sql`

3. **Jalankan aplikasi:**
   - Buka: http://localhost/PROJECT SQL/
   - Login dengan: `admin` / `admin123`

4. **Mulai gunakan!**
